# Full Stack Development — Hands-on Course

Welcome to the Full Stack Development course. This repository contains short, focused lessons and examples that introduce core web development concepts through practical, hands-on files you can open, edit, and run locally.

**What this course covers:**
- Fundamentals of HTML & CSS
- Building and previewing static pages
- Hands-on examples to modify and extend

**Who this is for:**
- Beginners learning web development basics
- Developers who want quick, practical examples

## Learning Outcomes
- Build simple HTML pages and style them with CSS
- Understand the structure of a basic front-end project
- Open and iterate on example files in a browser

## Project Structure
- Day_01/: Intro to HTML & CSS (examples and notes)
	- Concept_html.md — notes and explanations
	- intro.html — starter HTML example
	- intro.css — associated styles
- Day_02/: Follow-up exercises and notes
	- basic.md — summary and exercises

## How to Use
1. Open the files in your editor (VS Code recommended).
2. Preview `intro.html` in a browser. You can right-click and choose "Open with Live Server" if you have the Live Server extension.
3. Edit the HTML/CSS files and refresh the browser to see changes.

Recommended commands (optional):
```powershell
# If you have Python installed, a quick HTTP server:
python -m http.server 8000
# Then open http://localhost:8000/Day_01/intro.html
```

## Suggested Schedule
- Day 1 — Read `Concept_html.md`, open `intro.html`, experiment with `intro.css`.
- Day 2 — Read `basic.md`, complete exercises and extend the example pages.
